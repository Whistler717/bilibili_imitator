# bilibili_imitator

基于 **HarmonyOS（鸿蒙）ArkUI** 开发的 B站（哔哩哔哩）风格视频客户端 App，使用 ArkTS 语言编写。

## 功能特色

- **首页推荐**：轮播 Banner + 分类导航 + 视频卡片瀑布流，模拟 B站推荐页
- **分区浏览**：按分区（音乐、游戏、知识、美食等）筛选视频内容
- **视频播放**：支持本地视频播放，包含视频详情页、点赞/投币/收藏等交互
- **评论系统**：评论列表展示 + 评论输入功能
- **搜索功能**：视频搜索页面
- **动态页**：关注用户的动态信息流
- **用户系统**：登录/注册页面、个人主页、关注列表
- **历史与收藏**：观看历史记录、收藏夹管理
- **本地数据库**：基于 HarmonyOS 关系型数据库存储用户数据

## 技术栈

- **HarmonyOS NEXT** / ArkUI 声明式 UI
- **ArkTS** (TypeScript 超集)
- 关系型数据库（RelationalStore）
- 本地 Mock 数据（含真实视频文件用于演示）
